# **MAB (Mobilidade Acessível Brasileira)**

# **Descrição**

    Devido à ausência de um software de viagens que apresentasse a estrutura necessária para atender às necessidades dos PCDs (Pessoas com Deficiência), foi desenvolvido um aplicativo composto por motoristas e veículos adaptados no trabalho para com esses indivíduos, além de também possuir um sistema de fácil manuseamento.  

# **Objetivo**

    Fornecer viagens acessíveis aos PCDs (Pessoas com Deficiência).

# **Funções de cada integrante**

    Ana Paula - Classe Passageiro(Pessoa)
    Caio - Classe Viagem
    Diogo - Classe FormaPagamento
    José Júnior - Classe Motorista(Pessoa)
    Luiza - Classe Veiculo
    Marta - Superclasse Pessoa
    Yana - Classe Endereco
